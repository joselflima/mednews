# Specs

## Objective

This folder contains the specifications for the agents that will be used to develop the project.

## Folder structure

├── specs                   # Task specifications
│   └── <task_name>
│       ├── plan.md         # Plan of how to do the task
│       ├── spec.md         # Specification of the task to be done
│       ├── tasks.md        # Tasks to be done
│       └── report.md       # Report of what was done
